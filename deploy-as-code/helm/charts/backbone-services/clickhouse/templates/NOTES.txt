ClickStack has been installed.

By default, this chart deploys the full ClickStack platform: HyperDX, ClickHouse
(via the ClickHouse Operator), MongoDB (via the MongoDB Community Operator), and
the OpenTelemetry Collector. Each component can be disabled to use an external
instance instead.

To disable a built-in component, set its enabled flag to false:
  helm upgrade {{ .Release.Name }} <chart-name-or-path> \
    --set clickhouse.enabled=false \
    --set mongodb.enabled=false \
    --set otel-collector.enabled=false

When disabling a component, override the corresponding endpoints in hyperdx.config
to point at your external service (e.g. CLICKHOUSE_ENDPOINT, MONGO_URI).
